# 🔐 Password Generator

## 📌 Description
This project is a Password Generator developed using Python as part of the **CodeSoft Python Programming Internship**. It generates strong and random passwords based on the length provided by the user.

## ✨ Features
- Generates random passwords
- User-defined password length
- Includes uppercase letters
- Includes lowercase letters
- Includes numbers
- Includes special characters

## 🛠️ Technologies Used
- Python 3
- random module
- string module

## ▶️ How to Run
1. Clone this repository.
2. Run the Python file.
3. Enter the desired password length.
4. The program generates a secure random password.

## 📚 Concepts Used
- Python Modules
- Randomization
- Strings
- Loops
